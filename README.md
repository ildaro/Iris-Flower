## Introduction
The goal of this project is to investigate the famous Iris flower dataset and find interesting observations through the use of Python
and related tools. The Iris flower data set is a classic dataset that is a staple in the computing world when it comes to data science.
The data set was first introduced by the well-known statistician and biologist Ronald Fischer in 1936, it contains 50 samples each of 
the following 3 species of Iris flower: Iris setosa, Iris versicolor and Iris virginica.
